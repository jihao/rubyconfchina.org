2012 第四届中国 Ruby 大会

11月16日 - 11月17日 上海杨浦区创智天地

大会介绍

RubyConfChina 大会是一个为期两天的技术大会，由 Ruby China 和 TopGeek 社区共同发起。本年度会议将在是国内最高水平的 Ruby 大会，本次大会邀请到 Ruby 语言之父松本行弘先生，以及国内最顶尖的 Ruby 开发高手和大批的 Ruby 开发人才，并帮助国内使用 Ruby 技术的公司宣传，推广，并跟全球 Ruby 社区建立良好关系。

演讲嘉宾

演讲嘉宾：松本行弘（matz）

演讲主题：Ruby 2.0

嘉宾介绍：松本行弘（matz）是Ruby语言的发明人，他在1993年发布了Ruby语言的第一个版本，随后ruby语言逐渐在日本流行起来。

2000年，Dave Thomas出版了《Programming Ruby》一书，首次将这个发源于日本的动态编程语言介绍到了英文社区，Ruby开始在全球范围内流行。2004年，丹麦人DHH用ruby编写了Web快速开发框架Ruby on Rails，彻底掀起了Ruby语言的热潮。

演讲嘉宾：Fred Wu

演讲主题：如何通过努力和一些小聪明，成为一个更优秀的（ruby）程序员

嘉宾介绍：Fred Wu是一位对开源社区充满热情的程序员——为 Rails 等多个开源项目做过些许的贡献。出生和成长于上海，从13岁开始接触到并爱上了计算机与网络，从计算机杂志上“认识”了 HTML，萌发了对网络的憧憬。经过中学阶段对网络的痴迷到最终选择了互联网程序开发作为职业。Fred Wu毕业于澳大利亚墨尔本的 Monash 大学，就读于信息技术与系统（多媒体）专业。曾经在 Envato, PlayUp 和 SitePoint 等公司就职，目前在 Locomote 带领技术团队。

演讲嘉宾：Yi-Ting Cheng

嘉宾介绍：Yi-Ting Cheng (a.k.a xdite) is a web developer, Ruby on Rails specialist and popular blogger/entrepreneur in Taiwan, the author of beginner turtorial : "Rails 101" (http://rails-101.logdown.com).

Xdite has very deep knowledge of the full-stack website development & web operation, know to implement exciting idea in a amazingly short amount of time. Now her run a Rails consultant company living in Taiwan.

演讲嘉宾：蔡望勤

演讲主题：使用Rails/Sinatra实现企业智能能源云服务

嘉宾介绍：蔡望勤是China on Rails（chinaonrails.com）的创始人。1997年接触编程语言Pascal，2001年至2005年为珠三角多家知名企业研发知识管理系统和人力资源管理系统。2005年加入当时中国最大的社交网络平台UUZone，从此专注互联网应用研发。2007年赴北京参与创建时尚社交网络公司P1.cn，并担任首席技术官。现在他住在深圳，任易皆能科技(深圳)有限公司云服务研发总监。

主题介绍：Provide on-demand communications in order to establish better monitoring and control for Home Area Networks (HAN) devices.

演讲嘉宾：黄志敏

演讲主题一：JRuby @ OpenFeint

演讲主题二：Semi Automatic Code Review

嘉宾介绍：黄志敏 (a.k.a flyerhzm) 资深软件工程师，现就职于GREE，负责OpenFeint平台的基础设施，技术架构和服务器端性能优化。曾是2011年RubyKaigi和2012年Railsconf的演讲嘉宾，热衷于开源社区，也是bullet和rails_best_practices等多个开源项目的创建者和贡献者。

演讲嘉宾：马海波

演讲主题一：互联网创业，没有设计师，如何设计

演讲主题二：Rails 开源项目 之 ShopQi

嘉宾介绍：马海波是 ShopQi 云商店的创始人、一名全栈型程序员、开源爱好者。1997年迷上电脑游戏，开始学习电脑各种软硬件知识，在网吧谋得兼职，使用 VB 开发网吧管理工具；2004年毕业从事 Java 开发；2008年尝试使用 Ruby 做网站；2009年说服公司组建 Ruby 团队，开发企业应用；2010年学习 Object-C 开发 iphone 应用，做了个小游戏放到 App Store 卖，2011年开始创业。

演讲嘉宾：过纯中

演讲主题：定制Rails生成模板开发Web表单和Excel报告的工厂管理应用

嘉宾介绍：10年.NET经验，5个月Ruby On Rails经验，在SanDisk长期从事的上海工厂的生产管理软件运营开发和维护工作，自学和使用过很多微软技术，例如Silverlight, Winform, ASP.NET, WCF, WPF, Powershell等等，从自学ASP.NET MVC后决心转向Ruby on Rails平台，很高兴能以Rails 2012 开发者调查报告中占2%的Windows平台开发者身份参加第四届中国 Ruby 大会并分享Windows下的Rails开发经验。

演讲嘉宾：太檑

演讲主题：“How we test our projects

嘉宾介绍：Pragmatic.ly创始人。 Railscasts-china创始人。典型的理想主义者。热爱开源，喜欢分享，热衷于把自己知道的东西以各种方式分享给他人。对代码有洁癖，且有点偏执。从小励志成为职业玩家的他最终走向了技术宅的不归之路，最可怕的是他还深深的热爱上了这份事业。

演讲嘉宾：叶玎玎

演讲主题：Real Time Solutions with Ruby

嘉宾介绍：Pragmatic.ly创始人，前Intridea架构师，负责产品管理与开发。开源狂热者，喜欢解决各种疑难杂症，完美主义兼强迫症者。喜欢研究开发流程及方法论，关注如何改善团队协作。创业菜鸟，目标是做让中国开发者骄傲的产品。资深曼联球迷，防守型后腰，更关注如何让队友发挥的更好。

RubyChina社区组织

Ruby China Group 是一个非营利组织，它旨在为中国的 Ruby 和 Rails 爱好者提供一个自由，开放的交流平台。目前这里已经是国内最权威的 Ruby 社区，拥有国内所有资深的 Ruby 工程师。

TopGeek 顶尖极客沙龙

TopGeek 是面向技术极客的组织，定期举办线下活动，讨论软件和互联网的一些新的技术潮流，开发理念和经验。 这个组织不以语言或行业划分圈子，打破技术分割，开拓大家思路，促进互相学习，实现资源共享。解决一些成熟技术人员进一步发展的共同问题，促进交流和发展。口号："一日为极客，终身为极客！"
